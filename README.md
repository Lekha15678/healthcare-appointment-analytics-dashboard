# 🏥 Healthcare Appointment Analytics Dashboard

## 📌 Project Overview

The Healthcare Appointment Analytics Dashboard is a data analytics project developed using Python and Power BI to analyze patient appointment behavior and healthcare attendance patterns.

The project focuses on identifying factors affecting appointment attendance, patient demographics, waiting periods, and healthcare conditions through data cleaning, exploratory data analysis, and interactive dashboards.

---

## 🎯 Project Objectives

- Analyze patient appointment attendance.
- Identify factors affecting no-shows.
- Study the impact of SMS reminders.
- Analyze patient demographics and medical conditions.
- Develop an interactive Power BI dashboard for healthcare insights.

- Diabetes

- Handicap
- SMS Received
- No-Show Status

## 🧹 Data Preprocessing

The following preprocessing steps were performed:

- Removed invalid records.
- Converted date columns.
- Renamed inconsistent columns.
- Created Waiting Days feature.
- Removed negative waiting days.
- Feature engineering.
- Data validation.
